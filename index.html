Introduction: What is CSS?

The term "CSS" stands for "Cascading Style Sheets". It is a declarative language that applies structure to an html document. In order to achieve this through css selectors, properties and values are used to style the layout of the webpage.

Styling a webpage with properties and values is known as setting a CSS rule. Multiple CSS rules can be applied to change the design of a webpage layout. The page's appearence is set through the "cascading" of rules that govern how selectors are prioritized. CSS can be used alongside HTML and Javascript to make stunning webpages.

Selectors

"Selectors" designate the html that is being styled. Selectors can include html tags, IDs, and even style attributes.

    HTML tag example:

  • h1 {...}
    •

    ID example:

  • #header {...}
    •

    Style Attribute example:

  • style="..."
    •

    *** this way is actually referred to as an html attribute, however since it can affect the layout of a page, I have included in this documentation.

Properties

"Properties" designate how the selected elements are going to look. They can affect the width, height, color, etc of a webpage.

Example:

  • .header-style {
  • width: ...}
    •
Values

"Values" designate how much the selectors affect the selected elements. Values in css can be either numerical or non-numerical depending on the property.

Reset Templates

"Reset Templates" are code stylesheets that are used to overide the default template of internet browsers (i.e. safari, chrome, firefox, etc). Examples of reset templates are the Eric Meyers template, the Yahoo template, and the Trippoli template. There are many other templates that can be downloaded and used from the internet as well.
